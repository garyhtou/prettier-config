# `@garyhtou`'s Prettier Configurations

## Install

``` sh
yarn add -D @garyhtou/prettier-config
# or
npm install @garyhtou/prettier-config --save-dev
```

then add to `package.json`:

```json
{
	"prettier": "@garyhtou/prettier-config"
}
```
